# Jewelry Website (Shiny)

## Overview
This is a simple jewelry website created using HTML and CSS. It showcases different jewelry categories like rings, bangles, and necklaces.

## Objective
To design a clean and structured website while learning the basics of web development.

## Technologies Used
- HTML  
- CSS  

## Features
- Navigation bar  
- Dropdown menu  
- Product sections  
- Simple and clean layout  

## 💡 Learnings
- Improved HTML structure skills  
- Learned CSS styling and layout design  
- Understood how to organize a basic website  

## 🔧 Future Improvements 
- Improve UI design  
- Add more pages and features
